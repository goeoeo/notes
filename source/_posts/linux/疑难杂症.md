## ubuntu中解决机箱前置耳机没声音
1.安装pavucontrol软件：
 ```
 sudo apt-get install pavucontrol
 ```
2.选择配置-》模拟立体声双工

3.输出设备-》模拟耳机

## ubuntu 底部图标居中

```
sudo apt install cairo-dock
sudo apt install gnome-shell-extensions
sudo gnome-extensions disable ubuntu-dock@ubuntu.com
gnome-extensions enable ubuntu-dock@ubuntu.com
```
