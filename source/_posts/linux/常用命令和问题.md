# ubuntu20.04 频繁卡死重启
NVIDIA 独立显卡在ubuntu中使用其默认驱动，不兼容，导致频繁死机重启   
解决办法，为独立显卡安装官方驱动:  
1. ubuntu-drivers devices  #查询驱动
选择recommended 推荐驱动
![](常用命令和问题/img.png)
2. sudo apt install nvidia-driver-470 #安装驱动  
3. sudo reboot # 重启  
4. nvidia-smi # 查看显卡状态
